ICP-Based Euclidean Distance Measurement for Tunnel Deformation Analysis

Source: Dr. Mahamadou CAMARA, PhD Geomatics Engineering mahamadoucamaramoc@outlook.com

1. Introduction

This document presents the mathematical formulation of the Iterative Closest Point (ICP) concept and Euclidean distancebased deformation measurement, as implemented in the Python workflow for shield tunnel deformation analysis using mobile laser scanning point cloud data. In this study, geomatics engineering methods combined with mathematical modeling are applied, including mobile laser scanning (LiDAR) point cloud processing, ICP-based point correspondence, Euclidean distance measurement for three-dimensional displacement analysis, and geometric modeling (ellipse fitting and ovality assessment), to accurately quantify tunnel deformation.

2. Point Cloud Representation

Let the tunnel point clouds acquired at two different epochs be defined as: P = { p = (x, y, z) | i = 1, , N } P = { q = (x, y, z) | j = 1, , M } where P and P represent the tunnel surface at times t and t, respectively.

3. ICP Principle and Correspondence Search

The core objective of ICP is to minimize the distance between corresponding points in two point clouds. For each point p P, the closest point q P is identified by minimizing the Euclidean distance: q = arg min || p q || In this study, the correspondence search is efficiently implemented using a cKDTree structure.

4. Distance Threshold and Outlier Rejection

A tolerance threshold is introduced to discard unreliable correspondences: d Points exceeding this threshold are considered outliers and removed, improving robustness against noise and occlusions.

5. Euclidean Distance Deformation Equation

Once the closest point correspondence is established, the deformation magnitude d for each point is computed as:

d = [(x x) + (y y) + (z z)]

This Euclidean distance represents the true three-dimensional displacement of the tunnel lining.

6. Relevance to Tunnel Deformation Monitoring

The ICP-based correspondence and Euclidean distance framework enables millimeter-level deformation detection and is well suited for shield tunnel monitoring using high-density mobile laser scanning data.

# -------------------------------------------------
# ICP algorithm (translation only - simplified)
# -------------------------------------------------
def icp(A, B, max_iterations=50, tolerance=1e-5):
    src = np.copy(A)
    prev_error = 0

    for i in range(max_iterations):
        nbrs = NearestNeighbors(n_neighbors=1).fit(B)
        distances, indices = nbrs.kneighbors(src)

        T = np.mean(B[indices[:, 0]] - src, axis=0)
        src += T

        mean_error = np.mean(distances)
        if abs(prev_error - mean_error) < tolerance:
            break
        prev_error = mean_error

    return src

# Calculate Ovality and Axis Length Change

def fit_ellipse(points):
    pca = PCA(n_components=2)
    pca.fit(points)
    axis_lengths = np.sqrt(pca.explained_variance_)
    ovality = np.abs(axis_lengths[0] - axis_lengths[1])
    return ovality, axis_lengths

# Project the 3D points onto the XY plane for ellipse fitting
